# ASHRAE 170P HVAC Calculator — PWA (Mobile Web)

## تشغيل محلي (للتجربة)
> لازم يكون عبر سيرفر (مو فتح ملف مباشرة) عشان الـ Service Worker يشتغل.

### Python
python -m http.server 8000

ثم افتح:
http://localhost:8000

## نشر مجاني (الأبسط): GitHub Pages
1) أنشئ repo جديد على GitHub (Public)
2) ارفع محتويات هذا المجلد كما هي
3) من Settings > Pages اختر Branch: main و Folder: / (root)
4) راح يطلع لك رابط. افتحه من الجوال.

## تثبيت على iPhone (كأنه تطبيق)
Safari > Share > Add to Home Screen

## تثبيت على Android
Chrome > menu > Add to Home screen
