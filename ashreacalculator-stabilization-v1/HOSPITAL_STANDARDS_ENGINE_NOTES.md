# Hospital Standards Engine Package

## What this package adds
- Hospital room standards dataset grouped by categories
- Profiles for OR, ICU, Isolation, Emergency, Labs, Patient/Exam/Treatment, and non-clinical rooms
- Standards logic shown in result card
- Standards metadata saved in history
- Standards logic shown in quotation cards
- Standards logic shown in technical report

## Important
This package establishes the engine structure and draft room profiles.
It is meant to be the strong foundation before final value tuning and equipment load logic.
